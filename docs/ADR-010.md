### ADR-010: Dependency and Threat-Landscape Check, July 2026
Date: July 5, 2026
Decision: Checked ARGUS's actual code against current (2026) CVE disclosures for
LangGraph, PyMuPDF, and LangChain-core, and against the OWASP LLM/ASI Top 10 2026
update, rather than assuming the stack is safe because it was safe when chosen.
Context: A third-party pentest scan prompted a broader check, not just the headers it
found. Real, severe 2026 CVEs exist in this stack: CVE-2025-67644 (SQL injection,
LangGraph SQLite checkpointer) chained with CVE-2026-28277 (deserialization) to
achieve RCE on self-hosted servers, CVE-2026-27022 (same class, Redis checkpointer),
and CVE-2026-3029 (PyMuPDF path traversal via embedded-file extraction).
Verification, not assumption: `graph.py`'s `builder.compile()` has no `checkpointer`
argument, ARGUS never uses LangGraph's persistence layer, ruling out all three
checkpointer CVEs by direct code inspection. `document_processor.py` only calls
`fitz.open()` and `page.get_text()`, never the vulnerable `embedded_get`/embed-extract
function, ruling out the PyMuPDF CVE the same way. No `langchain_core` import or
`load_prompt()` call exists anywhere in the codebase, ruling out the related
LangChain-core CVEs. `pip-audit` run directly against the real pinned
`requirements.txt`: no known vulnerabilities found. `npm audit` on the frontend: one
moderate, low-exploitability transitive finding (postcss, bundled inside Next.js,
no user-controlled CSS generation exists in this app to trigger it), tool's suggested
fix would downgrade Next.js by seven major versions and was not applied.
Consequence: Real finding, not code, documentation: `BLUEPRINT.md`'s compliance
table marks LLM03:2025 (Supply Chain) closed, citing a pip-audit/npm audit CI gate.
That gate does not exist yet, no `.github/workflows/` directory exists in the repo,
it's Phase 5 scope. This ADR's manual check is a one-time stand-in for a control the
documentation already claims is live. Re-run manually on a regular cadence until
Phase 5 actually builds the automated gate.
Forward note, not urgent now: OWASP's 2026 LLM guidance names multimodal injection
(malicious instructions hidden in images) as a gap current text-only defenses can't
solve. Not exploitable against ARGUS today, PyMuPDF extracts text only, there is no
image-reading channel yet. Revisit this specifically if image or figure reading is
ever added, the current guards do not automatically extend to cover it.
Status: Accepted. No code changes required from this pass beyond what Sprint 2.2
already fixed independently (the classifier prompt). Findings logged in
CONTINUITY.md's gaps table.
